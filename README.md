# 공유니폼 사용 가이드

## 안드로이드 앱 사용법

### 1. APK 다운로드 및 설치
- `gonguniform.apk` 파일을 다운로드하여 안드로이드 기기에 설치합니다.

### 2. 앱 로그인
- 앱을 실행하고 다음 정보를 입력하여 로그인합니다:
  - **ID:** `kim@ewhain.net`
  - **Password:** `qwer1234`

### 3. 구단 선택
- 현재 작업 완료된 구단은 '두산'뿐입니다.
- 구단 선택 시 '두산'을 선택합니다.

### 4. 유니폼 선택 및 결제
- 아래의 순서로 선택합니다:
  1. `김재환`
  2. `밀리터리`
  3. `사이즈 L`
- `결제하기` 버튼을 클릭합니다.
